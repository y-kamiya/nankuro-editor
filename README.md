# nankuro-editor

A tool to help solving nankuro puzzles.

## Usage

https://y-kamiya.github.io/nankuro-editor/public/

Load a JSON file like the examples in the `examples` directory.

```json
{
  "data": [[1, 2, 3], ...],
  "answers": { "1": "ア", "2": "イ" }
}
```

## Development

### Requirements

- Node.js

### Setup

```bash
npm install
```

### Start dev server

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview build

```bash
npm run preview
```
